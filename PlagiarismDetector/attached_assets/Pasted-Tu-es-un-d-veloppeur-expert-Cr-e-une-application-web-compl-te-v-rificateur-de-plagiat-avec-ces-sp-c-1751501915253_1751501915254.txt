Tu es un développeur expert. Crée une application web complète vérificateur de plagiat avec ces spécifications : 1. Front-end Interface simple, claire, responsive, moderne. Grand champ texte pour coller le texte à analyser. Bouton "Vérifier le plagiat". Affichage du rapport avec : Pourcentage de texte copié, paraphrasé, original. Liste des passages détectés avec leur source (URL ou extrait). Recommandations pour améliorer le texte. Surlignage dans le texte soumis des passages détectés. Barre de progression pendant l’analyse. Bouton export PDF du rapport. Logo SVG simple (bouclier + loupe) dans le header. 2. Back-end Serveur Node.js + Express.js avec API REST sécurisée. Analyse du texte en segments (phrases/paragraphe). Détection plagiat simulée par corpus en mémoire + enrichie par appels à Google Custom Search Engine API (CSE API). Calcul de similarité simple entre segments soumis et résultats Google CSE. Retour JSON structuré détaillé. 3. Monétisation Intégration Google Ads avec bannière responsive. Chargement asynchrone respectant RGPD. 4. Gestion des clés API L’application utilise plusieurs clés / tokens réels : Google Ads ca-pub-xxxxxxxxxxxxxxx Google Custom Search API Key (GOOGLE_API_KEY) Google Custom Search Engine ID (GOOGLE_CSE_ID) Système interactif dans le code (front & back) : Dès qu’une clé API est nécessaire, le programme doit interrompre la génération et me demander explicitement la clé à fournir. Ces clés sont stockées dans un fichier .env sécurisé côté serveur. Le README explique comment créer un projet Google Cloud, activer l’API Custom Search, créer un moteur personnalisé et obtenir ces clés. Même chose pour Google AdSense (compte, validation, récupération du ca-pub-xxxx). Le code doit clairement séparer la configuration des clés du reste. 5. Bonus Tests unitaires simples (Jest). Page "À propos" expliquant l’application. Instructions pour lancer localement. Merci de générer tout le code complet, fonctionnel et organisé, avec ce système interactif de demande et gestion des vraies clés API Google Custom Search Engine et Google Ads, ainsi que le logo SVG, la barre de progression, le surlignage et l’export PDF.
